# Changelog
- Added the new Actions system for declarative ViewModel method binding.

  - Supports parameterized actions via custom binders.
  - Generates efficient auto-registered JS event handlers.

- Improved JS generation pipeline for better performance.
- Internal refactoring and code cleanup for stability and maintainability.
- Updated reference manual and developer documentation.
